The number of participants to show in Latest and Ranked can be set in the file database.php

The parameters to connect to the database can be set in database.php

To webpage to show the lateste runs: <url>/latest.html
To show the webpage with  ranked runs, the class can be selected with a query string appended to the URL:
<url>/ranked.html?class=<class>
for example http://<IP>/ranked.html?class=T1
