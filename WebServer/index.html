WebServer Index

Page Description
top12.html Displays the top 12 drivers and their finish times.
Ranked.html Shows ranked results per class with the best times and intervals.
RankedPersistent.html Displays persistent ranked results per class with the best times and intervals.
Latest.html Displays the latest runs with details like driver, car, and times.
LastClass.html Shows the leaderboard for the last class that ran.
top12.php Fetches the top 12 results from the database in JSON format.
getranked.php Fetches ranked results for a specific class from the database.
getlatest.php Fetches the latest runs from the database in JSON format.
getlastclass.php Fetches the leaderboard for the last class that ran from the database.
classes.html Displays a list of available classes and their ranked results on selection. Used for selecting which class is shown on RankedPersistent.html and useful for remote selection of graphics on the live timing screen.
